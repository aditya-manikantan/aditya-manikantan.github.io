How I improve performance

Playbook

I treat performance like a product feature: measure → fix the biggest bottleneck → verify.

  • Database: indexes, query shape, set-based updates, avoiding N+1, correct data types.
  • API: batching, idempotency keys, payload trimming, caching, resilient retries.
  • App: async + cancellation, resource disposal, reducing allocations, pagination.
  • Proof: dashboards/benchmarks to show before/after (latency, throughput, errors).
